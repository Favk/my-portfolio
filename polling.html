Django Polling Website

Project Overview

Developed a polling application during an internship with Aleric Technologies, following the official Django documentation. This project aimed to learn the fundamentals of Django while creating a functional web application.

Features:

  • User Authentication: Enabled secure user login and voting functionalities.
  • Dynamic Content: Allowed users to create polls, vote on options, and view real-time results.
  • Responsive Design: Utilized Bootstrap for CSS to ensure a responsive and visually appealing interface across various devices.

Technologies Used: Django, Python, HTML, CSS (Bootstrap), JavaScript.

Outcome:

This project provided a solid foundation in Django and web development, showcasing the ability to build and deploy a full-stack web application with a clean and user-friendly interface. It also demonstrated practical experience gained during the internship with Aleric Technologies.
